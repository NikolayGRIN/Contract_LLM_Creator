2.1 Поставка осуществляется Поставщиком в адрес Покупателя/в Склад Покупателя.
2.2 Срок поставки составляет 30 дней.
2.3 Начало течения срока поставки начинается с даты подписания договора.
2.4 График поставки согласовывается отдельно для каждой партии.
2.5 Частичная поставка разрешается по просьбе Покупателя.
2.6 Досрочная поставка разрешается по просьбе Покупателя.
2.7 Товары должны быть подготовлены к отгрузке и готовы к передаче.
2.8 Перевозка товара организуется Поставщиком.
2.9 Погрузка товара осуществляется за счет Поставщика.
2.10 Разгрузка товара осуществляется за счет Покупателя.
2.11 Переход рисков происходит в момент передачи товара Покупателю.
2.12 Переход права собственности происходит в момент передачи товара Покупателю.
2.13 Товары должны быть упакованы для сохранности при перевозке.
2.14 Товары должны быть маркированы для идентификации партии/единиц товара.
2.15 Поставка должна соответствовать согласованной спецификации.
2.16 Проверка количества товара осуществляется при приемке.
2.17 Проверка качества товара осуществляется при приемке.
2.18 Документы поставки составляются стандартными документами поставки.
2.19 Подписание приемочных документов происходит при передаче товара Покупателю.
2.20 Сроки осмотра/проверки определяются отдельно для каждой партии.
